ServiceNow

Since Camel 2.18

The ServiceNow component provides access to ServiceNow platform through their REST API.

From Camel 2.18.1 the component supports multiple version of ServiceNow platform with default to Helsinki. Supported version are API Mapping and API Mapping

Maven users will need to add the following dependency to their pom.xml for this component:

    <dependency>
        <groupId>org.apache.camel</groupId>
        <artifactId>camel-servicenow</artifactId>
        <version>${camel-version}</version>
    </dependency>

URI format

    servicenow://instanceName?[options]
